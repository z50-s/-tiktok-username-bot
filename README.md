# TikTok Username Bot

بوت تيليجرام للبحث عن يوزرات تيك توك النادرة المتاحة.

## المميزات
- البحث التلقائي المستمر
- الحفظ التلقائي لليوزرات
- إشعارات فورية
- تشغيل دائم 24/7

## النشر على Railway

1. انسخ هذا المستودع
2. أنشئ مشروع جديد على Railway
3. أضف متغير البيئة `BOT_TOKEN`
4. انتظر حتى ينتهي النشر

## الأوامر
- `/start` - بدء البوت
- `/quick` - بحث سريع
- `/auto_start` - بحث تلقائي
- `/auto_stop` - إيقاف البحث
- `/saved` - اليوزرات المحفوظة
- `/stats` - الإحصائيات
